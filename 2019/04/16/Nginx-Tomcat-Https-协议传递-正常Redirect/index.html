Nginx Tomcat Https 协议传递,正常Redirect

Nginx 配置

Nginx 在请求头里设置 X-Forwarded-Proto 值,传递给 Tomcat .Tomcat 通过 Valve 将值设置到 Request.scheme

org.apache.catalina.valves.RemoteIpValve

然后 Redirect 时 协议头就能保持一致了
